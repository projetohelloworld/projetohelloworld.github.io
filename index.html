O projeto Hello World tem como objetivo proporcionar aulas de lógica de programação para alunos do ensino médio das instituições da cidade de Pouso Alegre.

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991. Python é conhecida por sua sintaxe clara e legível, o que a torna uma escolha popular entre programadores iniciantes e experientes.

Para a execução do projeto foi proposto que a equipe desenvolvedora tivesse um nome, para isso foi decido o nome de ponto e Vírgula pois algumas linguagens como Java, C++, C# e PHP ao final de cada linha de código deve-se colocar ponto e vírgula para evitar assim erro de execução coisa que não acontece no python.

O projeto Hello World conta com um vasto acervo de materiais de apoio que irão ajudá-lo a aprofundar-se no mundo da programação, incluindo conteúdos como vídeos, podcasts, livros e atividades.
